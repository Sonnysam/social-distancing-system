## Social Distancing System
An Arduino system built with an ultrasonic sensor, LED etc. to check for safe distancing in hospital queues 

## ℹ️ Please find photos of project in Photos Folder 📁 

## Tools used 🚀
- Arduino Uno
- Ultrasonic sensor
- Bread board
- Jumper wires
- Red & Green LED
- Arduino IDE nightly
